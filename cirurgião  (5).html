🤖 6. Consultar IA

📖 Manual: O que é cada coisa?

function nome() { }
Uma ação que o código sabe fazer. Começa com function, tem um nome, e tudo que ela faz fica entre { }.
function salvarMemoria() { }
localStorage
É onde o cérebro do projeto fica salvo no navegador. Mesmo fechando, os dados ficam lá.
localStorage.setItem("dados", valor);
fetch() / API / Gemini / OpenAI
É a parte que conversa com a IA. fetch faz uma chamada para a internet. Geralmente tem uma URL com api e uma chave secreta (apiKey).
import / export / require
São as conexões entre arquivos. import traz algo de outro arquivo. Em código embolado isso pode estar quebrado.
🚨 O que é vínculo Replit?
São partes do código que só funcionam dentro da Replit. Quando você para de pagar, o app para de funcionar. Exemplos:

replit.com · /api/code-assistant · /api/tts · REPL_ID · REPLIT_DB
